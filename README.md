## 前言

随着人们生活水平的提高，环保意识逐渐加强，垃圾分类成为了一种新时尚。为了更好地普及垃圾分类知识，提高大家的分类投放意识，我们开发了一款名为"垃圾分类小程序SSM"的应用。以下是该项目的详细介绍。

## 内容介绍

垃圾分类小程序SSM是一款基于Java语言的微信小程序，主要功能包括：垃圾分类知识普及、垃圾分类查询、投放记录等。用户可以通过该小程序方便快捷地了解垃圾分类知识，提高垃圾分类投放的正确率。

## 技术介绍

- 语言：Java
- 使用框架：Spring、Springmvc、MyBatis、微信小程序
- 前端技术：JS、Vue、CSS3、Uniapp

- 数据库：MySQL 5.7/8.0

- Maven：apache-maven 3.8.1-bin
